About

WiredIn

WiredIn was the second project I worked on at the KU Coding Bootcamp. Unfortunately, this group project is mostly a proof of concept rather than a functional app. I've chosen to showcase it regardless, as, despite being unfinished, it is a showcase of teamwork. WiredIn, in concept, is a social media for gamers. The idea stemmed from book clubs - people bond over how many books they have to read, so why not bond over how many video games you have to play? The site can be demoed, but it lacks most functions.

Check it out here

Catch-All Music Recommender

Catch-All was the first group project I completed at the KU Coding Bootcamp. It's a simple web-based app that takes an artist's name via user input and returns the top 3 most similar artists using the last.fm API. Using the See More button, the user can get information about the artist including a bio, user tags, and the top 3 results for the artist on Discogs. The app is lightweight and can be added to easily. It was a group project, though I handled all of the JavaScript.

Check it out here
